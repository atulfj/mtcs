REQUIREMENTS
	Python >= 3.8.10
	NumPy >= 1.17.4
	VTK >= 9.2.5
	
EXECUTION GUIDELINES
	1. For isocon.py, make sure the data is present in the path "Data/Isabel_2D.vti".
	
	2. For volume.py, make sure the data is present in the path "Data/Isabel_3D.vti".
	
	3. Run in command line using
	
			python3 isocon.py
			
				and 
				
			python3 volume.py
